AUTO-CREDIT: `processMaturedFDs(userId)` — for manual FDs past maturityDate with maturedCredited!=true: flips maturedCredited FIRST (idempotency), then posts a CONFIRMED income credit (merchant "FD Maturity: <name>", category income) to creditAccountId via createTransaction → updates that account balance.
